# 端口可视化管理器

一个美观实用的桌面端口可视化管理工具，采用透明磨砂玻璃效果设计，用于实时监控和管理系统中正在监听的端口。

![Electron](https://img.shields.io/badge/Electron-28.0.0-47848F?logo=electron)
![Platform](https://img.shields.io/badge/Platform-Windows-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ 核心功能

### 🎨 界面特性
- **透明磨砂效果** - 采用 `backdrop-filter` 技术，呈现优雅的磨砂玻璃质感
- **深色主题** - 舒适的深色半透明设计，不影响桌面内容观察
- **圆角窗口** - 现代化 16px 圆角设计
- **自定义标题栏** - 无边框设计，完全自定义的窗口控制
- **流畅动画** - 悬停效果、加载动画等细节优化

### 📊 监控功能
- **实时端口监控** - 自动扫描并显示所有正在监听的端口
- **进程信息展示** - 显示每个端口对应的进程名称和 PID
- **自动刷新** - 支持手动刷新和 3 秒自动刷新
- **搜索过滤** - 按端口号、进程名或 PID 快速搜索

### 🔍 智能识别

#### Web 服务检测
自动识别 HTTP/HTTPS 服务，支持以下端口：
- 标准端口：80, 443
- 开发端口：3000-9000 范围内的 Web 服务
- 常见框架端口：8080, 8000, 8888, 4200, 5000, 5173, 5174, 8081, 8443, 9000

**功能**：
- 🔵 蓝色地球图标标识
- 点击可直接在浏览器中打开网页
- 悬停显示完整 URL

#### 数据库服务识别
自动识别 30+ 种数据库服务：

**关系型数据库**
- MySQL (3306, 3307)
- PostgreSQL (5432, 5433)
- SQL Server (1433, 1434)
- Oracle (1521)
- CockroachDB (26257, 26258)

**NoSQL 数据库**
- MongoDB (27017-28017)
- Redis (6379-6381)
- Elasticsearch (9200, 9300)
- Cassandra (7000, 9042, 9160)
- CouchDB (5984)
- Neo4j (7474, 7687)
- ArangoDB (8529)
- InfluxDB (8083, 8086)

**消息队列**
- RabbitMQ (5672, 15672)
- Kafka (9092)
- ZooKeeper (2181)

**其他**
- Memcached (11211)

**功能**：
- 🟢 绿色数据库徽章标识
- 显示数据库类型名称
- 数据库图标直观展示

#### 系统进程标记
自动识别 Windows 系统关键进程：
- 核心系统：System, svchost.exe, csrss.exe, wininit.exe, winlogon.exe
- 系统服务：services.exe, lsass.exe, smss.exe
- 桌面进程：explorer.exe, dwm.exe
- 其他系统组件：taskhostw.exe, spoolsv.exe, conhost.exe 等

**功能**：
- 🟠 橙色"系统"徽章标识
- 关闭时显示额外警告提示
- 浅橙色背景高亮显示

### ⚙️ 管理功能
- **一键关闭端口** - 强制终止占用端口的进程
- **安全确认** - 关闭操作需二次确认
- **系统进程警告** - 关闭系统进程时显示特殊警告
- **权限提示** - 需要管理员权限时会提示

## ⚠️ 注意事项

### 安全警告
1. **谨慎关闭端口** - 强制终止进程可能导致数据丢失
2. **系统进程** - 关闭系统进程可能导致系统不稳定或崩溃
3. **数据库进程** - 关闭数据库进程可能导致未保存数据丢失
4. **权限要求** - 某些进程需要管理员权限才能关闭

### 使用说明
- 遇到权限问题时，请以管理员身份运行 `start.bat`
- 如果端口列表为空，请检查是否有端口正在监听
- 自动刷新会每 3 秒更新一次，可能影响性能
- Web 服务检测需要网络连接

### 已知限制
- 仅支持 Windows 系统
- Web 服务检测可能有延迟（最多 1.5 秒）
- 某些隐藏进程可能无法识别
- 部分端口可能无法关闭（需要更高权限）

## 🎨 自定义配置

### 修改窗口大小
编辑 `main.js` 第 213-214 行：
```javascript
width: 450,   // 窗口宽度
height: 650,  // 窗口高度
```

### 修改自动刷新间隔
编辑 `public/app.js` 第 173 行：
```javascript
autoRefreshInterval = setInterval(fetchPorts, 3000); // 毫秒
```

### 添加数据库端口
编辑 `main.js` 中的 `DATABASE_PORTS` 对象：
```javascript
const DATABASE_PORTS = {
    3306: 'MySQL',
    // 添加你的端口
    1234: 'Your Database',
};
```

### 调整透明度
编辑 `public/style.css` 中的背景色：
```css
background: rgba(25, 25, 35, 0.8); /* 最后一个数字是透明度 */
```
